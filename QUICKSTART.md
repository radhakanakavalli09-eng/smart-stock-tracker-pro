# 🚀 Quick Start Guide - Stock Market Live Tracker

## Ready to Launch! 🎯

Your modern Stock Market Tracker application is fully set up and ready to use.

## ⚡ Ultra-Quick Setup (2 minutes)

### 1️⃣ Install Dependencies
Open PowerShell in this folder and run:

```bash
pip install -r requirements.txt
```

### 2️⃣ Launch Application
```bash
python modern_stock_tracker.py
```

**That's it!** The app opens automatically. ✅

---

## 🎮 First Time Usage

### Step 1: Add Your First Stock
1. Click the "Select Stock:" dropdown
2. Type or select **"Apple"** (or any company)
3. Click **"➕ Add Stock"**
4. Watch the data appear! 📊

### Step 2: Set a Price Alert
1. Click **🔔 Alert** button next to a stock
2. Set when to alert you:
   - **Alert if price goes ABOVE:** (e.g., $200)
   - **Alert if price goes BELOW:** (e.g., $150)
3. Click **Save Alert**

### Step 3: View a Chart
1. Click **📊 Chart** button next to a stock
2. See 5-day price trend
3. Close the window to return

### Step 4: Refresh Data
- **Auto-refreshes** automatically every 25 seconds
- Or click **🔄 Refresh Now** for instant update

## ❌ Something Not Working?

### Not Starting?
```bash
# Update packages
pip install --upgrade PyQt5 yfinance

# Try again
python modern_stock_tracker.py
```

### No Stock Data?
- ✅ Check internet connection
- ✅ Try a different stock (Apple, Tesla work great)
- ✅ Restart the app
- ✅ Some stocks have no data on weekends

### Alerts Not Showing?
- Notification popups always appear
- Desktop notifications need notifications enabled on Windows
- Still works - you'll see the popup!
